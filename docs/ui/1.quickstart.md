# Quick Start

## Installation

Install the package using pnpm:

```bash
pnpm add @ghentcdh/ui
```

## Configuration

Add the following to your `style.css` to enable Tailwind CSS, DaisyUI with the UGent theme, and the custom UI components:

```css
@import "tailwindcss";

@plugin "daisyui" {
  themes: ugent --default;
}
@import '@ghentcdh/ui/index.css';
```
